# Terraform Module - AWS S3 Bucket

Terraform module for creating an AWS S3 bucket with needed IAM permissions

## Usage

```
module "name_here" {
    source      = "git@github.com:sch3p/tfmod-s3//Module/?ref=v1.0"
    bucket_name = "bucket_name_here"
}
```
